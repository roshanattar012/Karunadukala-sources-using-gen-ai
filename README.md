# .Karunada-Kala Source

A production-ready Android application dedicated to discovering and preserving the cultural heritage of Karnataka. This app provides a bridge between traditional artisans and the modern digital audience.

## Features

- **Heritage Map**: Interactive map with location markers for Workshops (Red) and Performances (Yellow).
- **Artisan Directory**: Detailed profiles of Karnataka's master craftsmen and performers.
- **Smart Call**: One-tap phone connection to artisans via system dialer.
- **AI Cultural Context**: Integrated Google Gemini AI to provide real-time historical context for various art forms.
- **Rich Media Feed**: ExoPlayer-driven video feed showcasing traditional performances like Yakshagana and Dollu Kunitha.
- **Bilingual Support**: Full native support for both English and Kannada.

## Prerequisites

- **Android Studio Iguana** or newer.
- **JDK 17** (standard for modern Android development).
- **Google Maps API Key**: Enable "Maps SDK for Android" in Google Cloud Console.
- **Gemini API Key**: Obtain a key from [Google AI Studio](https://aistudio.google.com/).
